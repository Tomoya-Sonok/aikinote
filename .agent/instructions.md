# Agent System Instructions

このファイルは AikiNote プロジェクトの専属開発エージェントとして動く AI が最初に参照すべき指示書です。
あなたの行動と知識は、以下の `.agent/rules/` 配下のファイル群によって厳格に定義されています。

**いかなるタスクを実行する際も、回答生成前に必ず以下のステップを踏んでください。**

## 1. 必須プロトコルのロード (Critical)
ユーザーの指示に応答する前に、必ず **`.agent/rules/01_core_protocol.md`** を読み込んでください。
- そこに記載されている「思考プロセス(Analysis & Planning)」を実行してから回答を作成してください。
- 特に「日本語での応答」「SPファースト」の原則は絶対です。
- **TaskArtifact や ImplementationPlan などの成果物もすべて日本語で作成してください。**

## 2. 知識の参照 
タスクの内容に応じて、**`.agent/rules/02_project_knowledge.md`** を参照してください。
- 実装コードを書く場合 → 「ディレクトリ構造」「命名規則」を確認する。
- テストを書く場合 → 「AAAパターン」「Vitest設定」を確認する。
- コマンドを実行する場合 → 「開発コマンド集」を確認する。
- コミットを行う場合 → 「Git & コミット運用」を確認する。

## 3. エラー防止
あなたはルールファイルの内容と矛盾する回答をしてはいけません。
判断に迷った場合は、勝手に推測せず、ユーザーに確認してください。
ルールファイル同士で内容が競合している場合も、ルールファイルが陳腐化していて改善が必要な合図なので、ユーザーに確認してルールファイルを更新するよう働きかけてください。
