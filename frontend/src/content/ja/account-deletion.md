# アカウント削除リクエスト

本ページでは、AikiNote におけるアカウントおよび関連データの削除をリクエストする手順と、削除・保持されるデータの範囲についてご案内します。

## 1. 削除リクエストの手順

現時点では、AikiNote の iOS / Android アプリおよびウェブ版にはユーザー自身でアカウントを削除する UI は実装されていません。削除をご希望の方は以下の手順でリクエストを送信してください。

1. [AikiNote お問い合わせフォーム](https://docs.google.com/forms/d/e/1FAIpQLSfr11mzmwzwwXXULuoT4w8D57e9aAtUZa_9i8HDGAtDgjNxYw/viewform?usp=dialog) を開きます
2. 件名（または冒頭）に **「アカウント削除希望」** と明記します
3. 本文に以下を記入してください:
   - AikiNote に登録しているメールアドレス
   - 登録ユーザー名（任意）
4. フォームを送信します

※ お問い合わせフォームの代わりに、サポートメールアドレス **support@aikinote.com** へ同様の内容を直接ご連絡いただくことも可能です。

運営者側でリクエスト内容とアカウントの一致を確認したうえで、**受領後 7 営業日以内** に該当アカウントと関連データを削除します。処理完了後、ご登録のメールアドレス宛に完了通知をお送りします。

## 2. 削除されるデータ

アカウント削除リクエストに対して、以下のデータを削除します。

- プロフィール情報（ユーザー名、表示名、プロフィール画像、自己紹介文）
- 認証情報（メールアドレス、Supabase Auth のレコード、OAuth 連携情報）
- 稽古ページ（「ひとりで」機能で作成したすべてのページと添付画像・動画）
- 投稿（「みんなで」機能で投稿したすべての投稿および返信）
- タグ、カテゴリ設定、フォントサイズ設定などの個人設定
- 未読通知、プッシュ通知トークン、検索履歴
- S3 / CloudFront 上に保存されたすべての画像・動画ファイル

## 3. 削除されずに保持されるデータ

以下のデータは法令遵守・不正利用防止・会計義務のため、アカウント削除後も一定期間保持します。

| データ種別                                                               | 保持期間 | 理由                                     |
| ------------------------------------------------------------------------ | -------- | ---------------------------------------- |
| 決済・サブスクリプション履歴（Stripe / RevenueCat 連携分）               | 7 年間   | 電子帳簿保存法・法人税法等による保存義務 |
| 不正利用の記録・アクセスログ                                             | 1 年間   | 不正利用の調査および再発防止             |
| 本人確認のためにお問い合わせフォーム経由で頂いた削除リクエスト自体の記録 | 2 年間   | 削除依頼が正当に実施されたことの証跡     |

上記の保持データは、原則としてユーザー本人を特定できない形（匿名化・集計化）で保管されるか、必要最小限の範囲で暗号化して保管されます。
