# AikiNote プライバシーポリシー

最終更新日: 2026年3月4日

AikiNote（以下「本サービス」）は、ユーザーの個人情報保護を重要視しています。本プライバシーポリシーは、本サービスにおける個人情報の取り扱いについて定めるものです。

## 1. 収集する情報

本サービスでは、以下の情報を収集します。

### 1.1 ユーザーが提供する情報

- **アカウント情報**: メールアドレス、ユーザー名、パスワード（ハッシュ化して保存）
- **プロフィール情報**: プロフィール画像、所属道場名、合気道開始時期
- **稽古記録**: タイトル、稽古内容、コメント、タグ、添付ファイル（画像・動画）

### 1.2 自動的に収集される情報

- **アクセスログ**: IPアドレス、ブラウザ種類、アクセス日時
- **Cookie情報**: セッション管理および認証のために使用
- **利用状況データ**: ページ閲覧状況、機能の利用頻度等（個人を特定しない統計データとして、アクセス解析ツール「Umami」を通じて収集）

## 2. 情報の利用目的

収集した情報は、以下の目的で利用します。

1. 本サービスの提供、維持、改善
2. ユーザーアカウントの管理および認証
3. ユーザーサポートの提供
4. 本サービスに関するお知らせの送付
5. 不正行為の防止およびセキュリティの確保
6. 統計データの作成（個人を特定できない形で）

## 3. 情報の共有と開示

本サービスは、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

1. ユーザーの同意がある場合
2. 法令に基づく場合
3. 人の生命、身体または財産の保護のために必要がある場合
4. 本サービスの運営上必要な範囲で、業務委託先に提供する場合（適切な契約の下で管理）

## 4. 情報の保管とセキュリティ

1. ユーザーの個人情報は、セキュリティの高いクラウドサーバーに保管します。
2. データの送受信にはSSL/TLS暗号化通信を使用します。
3. パスワードは不可逆なハッシュ化を施して保存し、平文では保持しません。
4. 運営者は、個人情報への不正アクセス、漏洩、改ざん、紛失を防止するため、合理的な安全管理措置を講じます。

## 5. Cookieの使用

本サービスは、セッション管理および認証のためにCookieを使用します。ブラウザの設定によりCookieを無効にすることも可能ですが、その場合本サービスの一部機能が利用できなくなる場合があります。

## 6. 外部サービスの利用

本サービスでは、サービスの提供・運営のために以下の外部サービスを利用しています。各サービスにおける情報の取り扱いについては、各社のプライバシーポリシーもあわせてご確認ください。

| サービス名 | 提供元 | 利用目的 |
|---|---|---|
| **Supabase** | Supabase, Inc. | ユーザー認証およびデータベース管理 |
| **AWS S3 / CloudFront** | Amazon Web Services, Inc. | 画像・動画データの保存および配信 |
| **Umami** | Umami Software, Inc. | アクセス解析（Cookieを使用しないプライバシー配慮型の利用状況分析） |
| **YouTube API** | Google LLC | ユーザーが入力したYouTube動画URLのプレビュー情報（OGP）の取得・表示 |
| **Google認証** | Google LLC | ソーシャルログイン機能 |
| **Cloudflare** | Cloudflare, Inc. | CDN（コンテンツ配信）およびセキュリティ |

これらの外部サービスに対しては、本サービスの運営に必要な最小限の情報のみを提供し、適切な契約・設定のもとで管理しています。

## 7. ユーザーの権利

ユーザーは、以下の権利を行使することができます。

1. **アクセス権**: 保有する個人情報の開示を請求する権利
2. **訂正権**: 個人情報の訂正、追加または削除を請求する権利
3. **削除権**: アカウントの削除を請求する権利
4. **データポータビリティ**: 個人データの書き出しを請求する権利

これらの権利を行使する場合は、本サービス内のヘルプ・問い合わせフォームよりご連絡ください。

## 8. 未成年者の利用

本サービスは、特定の年齢制限を設けていませんが、未成年者が利用する場合は保護者の同意を得ることを推奨します。

## 9. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更することがあります。重要な変更がある場合は、本サービス内またはメールでお知らせします。変更後のプライバシーポリシーは、本ページに掲載した時点から効力を生じるものとします。

## 10. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、本サービス内のヘルプ・問い合わせフォームよりお願いいたします。
